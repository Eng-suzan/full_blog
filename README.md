"# full_blog" 
                      Full Blog System

A full-featured Blog Management System built using PHP & MySQL, allowing users to create, manage, and organize blog posts with image uploads and authentication system.

features
🔐 User Authentication (Register / Login / Logout)
📝 Create, Read, Update, Delete (CRUD) Blogs
🖼️ Image Upload with Posts
👤 User-based blog ownership
💬 Flash messages (success & error alerts)
🗂️ Organized project structure (MVC style)
🗑️ Automatic image deletion on post remove
🔒 Secure password hashing
